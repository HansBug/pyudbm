安装
===================

pyfcstm 目前托管在 PyPI 上，需要 Python >= 3.7，并且已在 CPython 3.7 到 3.14 上完成测试。

你可以使用以下命令从 PyPI 简单安装 pyfcstm：

.. code:: shell

    pip install pyfcstm

你也可以通过 GitHub 安装最新版本：

.. code:: shell

    pip install -U git+https://github.com/hansbug/pyfcstm@main
